Getting Started

  • Note: This website is still under development, but it still works and shows a proof of concept described below
  • See Github page for the development code
  • Please register and login
  • The objective of this site is to search for interesting stocks. You can create a custom stock filter.
  • For example check out the data set at Data Set 1 : Sort by target column for this table. All the rows labeled as Buy are stock that resulted in a minimum of 10% profit over 6 months. Rows with target variable as Sell are charts that resulted in a 10% loss. The goal is to develop a computational approach to recognize these trades.
  • Filter the stocks that resulted in a profit, Most of the charts show some kind of pattern. Identify these patterns/features
  • Are there any recurring patterns in the charts for the buy trades? If you find any recurring pattern, try representing it using stock price data and
  • You can then code up the features. Or just the built-in ones available
  • Once coded up, all the features are computed for all stocks everyday at midnight
  • You can create a filter/query using any feature or all of them
  • Save your filters/query and your filter will automatically run everyday, the results of which are shown on your homepage
  • You can label/categorize each chart, which are then used in a classification algorithm
